
Trabalho desenvolvido no Sistema Operacional Windows 10 pelos alunos Ricardo Ohara e Thiago Bucalão do curso Ciência da Computação da disciplina 6895-2 Arquitetura e Organização de Computadores II

Para poder executar o código em C# é necessário:

1 - Instalar o Visual Studio 2017 através do link:
	https://docs.microsoft.com/pt-br/visualstudio/install/install-visual-studio

2 - Baixe a versão vs_community.exe (Visual Studio Community)

3 - Necessário ter instalado NET.FRAMEWORK

4 - Instalado o Visual Studio, abra o programa

5 - Em Arquivo -> Abrir -> Projeto/Solução

6 - Selecione o arquivo "TomasuloConsole.sln"

7- Em Depurar -> Iniciar Depuração

8 - Irá abrir uma nova janela, com a seguinte frase: "Informe o diretório do arquivo 'Assembly':". Por exemplo, "C:\Users\Usuario\Tomasulo\Assembly.txt";

9- Para ver o próximo ciclo, basta apertar o botão "<ENTER>".

===============================================================================================================================================================================================
Por linha de comando (CMD)

1 - Acesse o diretorio onde se encontra o projeto. Por exemplo:
	cd TomasuloConsoleApp
	cd bin
	cd Debug
	cd TomasuloConsoleApp.exe

2 - Informe o diretorio onde se encontra o arquivo Assembly
